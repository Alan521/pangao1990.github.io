Python使用内置urllib模块或第三方库requests访问网络资源

发表于 2019-04-12 | 更新于 2019-04-22 | 分类于 Python | 评论数： | 热度： ℃
本文字数： 6.1k | 阅读时长 ≈ 10 分钟

Python 访问网络资源有很多方法，urllib, urllib2, urllib3, httplib, httplib2, requests ，现介绍如下两种方法：

  • 内置的 urllib 模块
    • 优点：自带模块，无需额外下载第三方库
    • 缺点：操作繁琐，缺少高级功能
  • 第三方库 requests
    • 优点：处理URL资源特别方便
    • 缺点：需要下载安装第三方库

添加Headers

通过 urllib 发起的请求会有默认的一个Headers：”User-Agent”:”Python-urllib/3.6”，指明请求是由 urllib 发送的。
